Alineación de Texto en CSS

Diferentes métodos para centrar texto y elementos

Métodos para Centrar Texto

1. text-align: center

El método más común para centrar texto horizontalmente dentro de un elemento de bloque.

.elemento { text-align: center; }

Este texto está centrado con text-align: center

2. Flexbox

Usando Flexbox para centrar tanto horizontal como verticalmente.

.contenedor { display: flex; justify-content: center; /* Centrado horizontal */ align-items: center; /* Centrado vertical */ }

Texto centrado con Flexbox

3. CSS Grid

Usando CSS Grid para centrar contenido en ambas direcciones.

.contenedor { display: grid; place-items: center; }

Texto centrado con CSS Grid

4. Margen Automático

Centrando un elemento de bloque horizontalmente con margen automático.

.elemento { width: 80%; margin: 0 auto; }

Elemento centrado con margen automático

5. Transformación Translate

Centrado exacto en el centro de un contenedor (útil para centrado vertical y horizontal).

.contenedor { position: relative; } .elemento { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }

Texto centrado con transformación

Ejemplos Prácticos

Centrado de Botones

Para centrar botones u otros elementos en línea, puedes usar text-align: center en su contenedor.

.contenedor-botones { text-align: center; }
Botón Primario Botón Secundario

Centrado Vertical y Horizontal

Para centrar contenido en el medio de la página, puedes combinar Flexbox con altura completa.

.pagina-completa { min-height: 100vh; display: flex; justify-content: center; align-items: center; }

Contenido centrado vertical y horizontalmente
